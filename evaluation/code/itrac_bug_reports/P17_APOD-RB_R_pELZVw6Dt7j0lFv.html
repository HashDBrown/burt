Bug summary:

Searching iTunes for "mis" causes an exception and displays no results.

Observed Behavior (a description of what happened with the app):

When attempting to add a podcast via searching iTunes and entering "mis" into the search field, the window displays the message "org.json.JSONException: No value for feedUrl" and a "Retry" button rather than displaying results.

Expected Behavior (a description of what you expected to happen):

The window should display search results for podcasts from iTunes that match "mis" as the search term.

Steps to Reproduce (the steps to reproduce the problem):

1. Open the app. Open the navigation menu if it does not open itself.

2. Click "Add Podcast," then "SEARCH ITUNES."

3. Click the magnifying glass in the top right to type in a search term. Type in "mis" and click the magnifying glass in the bottom right to search. The error message and Retry button will be displayed.

Additional information:

Unable to reproduce when searching for terms other than "mis" or when searching outside of iTunes.
