Bug summary:

Balance sheet duplicated after hitting back from settings

Observed Behavior (a description of what happened with the app):

I opened the left sidebar menu and went to check out reports (specifically, sheets). Then, I went back to the left sidebar to check settings, clicked nothing on settings, and instead hit the back arrow in the top left corner to return to the sheets report and suddenly all the lines inside the assets, liabilities, and equities were doubled (duplicated). For example, there were two totals in bold for each section.

Expected Behavior (a description of what you expected to happen):

I expected to hit back from the settings page and restore the same (non-duplicated) sheets page I had seen seconds earlier.

Steps to Reproduce (the steps to reproduce the problem):

1. Open app

2. Set up selecting: Next, USD, setup default accounts, send crash reports, Done, then Allow

3. I looked at the landing page a few seconds but didn't click anything and then hit the left hand menu (widget in the top left)

4. Select reports

5. Select sheets

6. Select the left hand menu once again

7. Select settings

8. Hit back (top left) to leave settings

9. You should now have the same duplicated bug!
