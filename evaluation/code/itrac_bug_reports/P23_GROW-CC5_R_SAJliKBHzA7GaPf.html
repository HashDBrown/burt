Bug summary:

The app crashes upon update of feed.

Observed Behavior (a description of what happened with the app):

When in the feed menu under a particular plant, if the user attempts to add an additive, the application will crash on completion.

Expected Behavior (a description of what you expected to happen):

The expected behavior is the additive will be included in the plant details.

Steps to Reproduce (the steps to reproduce the problem):

1. Open the application

2. Create a new plant

3. Name the plant and the type

4. Select the plant

5. Select the "Feed" menu

6. Select the "Add Additive" button

7. Add the additive and the mL

8. Select "Ok"
