Bug summary:

Added token does not appear in list

Observed Behavior (a description of what happened with the app):

User adds a new token, enters a name, generates a random seed and presses complete. Prompt says "Token Added", but the new token does not appear in the list.

Expected Behavior (a description of what you expected to happen):

The added token should appear in the token list screen.

Steps to Reproduce (the steps to reproduce the problem):

1. Tap 3 dotted option button.

2. Tap "Add Token".

3. Enter name for token in "Name" field.

4. Tap "Next".

5. Select "Generate Random Seed".

6. Tap "Complete".
