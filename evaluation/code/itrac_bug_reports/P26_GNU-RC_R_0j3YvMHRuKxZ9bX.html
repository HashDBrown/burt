Bug summary:

Using an apostrophe in the name of an a newly added account crashes the app

Observed Behavior (a description of what happened with the app):

When I attempt to name a new account with a name that contains an apostrophe, the app crashes.

Expected Behavior (a description of what you expected to happen):

I expect the account to be created with a name that contains an apostrophe.

Steps to Reproduce (the steps to reproduce the problem):

1. Click the "Create Accounts" button in the popup

2. Click the + button at the top right corner of the app

3. Type an apostrophe into the account name field and click the "Save" button
