Bug summary:

App crashes when trying to calculate BMI

Observed Behavior (a description of what happened with the app):

After opening the BMI Calculator and clicking on the Go button, the application crashes. This crash occurs even when the height and weight values are both entered. It also happens regardless of the number of weights previously entered in the app.

Expected Behavior (a description of what you expected to happen):

BMI should be correctly calculated and displayed based on the entered height and weight values.

Steps to Reproduce (the steps to reproduce the problem):

1. Open the DroidWeight application

2. Go to the "Statistics" page

3. Click on the "BMI Calculator" button

4. (Optional) Enter the height and weight values.

5. Click on the "Go" button
