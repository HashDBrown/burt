Bug summary:

App crashes when single quote character is used in the account name

Observed Behavior (a description of what happened with the app):

When creating a new account, if the single quote character (') is included in the account name, the application crashes. This crash occurs regardless of the selected currency and the existing accounts. Note: The app is able to handle all the other special characters.

Expected Behavior (a description of what you expected to happen):

The new account should be created with the single quote character(s) in its name and should be displayed in the list of existing accounts.

Steps to Reproduce (the steps to reproduce the problem):

1. Open the GnuCash application

2. Click on "Create Accounts" on the "Select accounts to create" window (any accounts selection is fine)

3. Click on the plus icon on the top right

4. On the "Create Account" pop up window, enter an account name that has at least one single quote character in it

5. Click on the "Save" button
