Bug summary:

app closes when trying to duplicate plant profile

Observed Behavior (a description of what happened with the app):

the app prompts for user input in order to create a plant profile. Later when the profile is replicated the app closes and on a second attempt at replicating the profile the app crashes

Expected Behavior (a description of what you expected to happen):

the expected behavior once duplicate is hit, is to make a copy of the already created profile

Steps to Reproduce (the steps to reproduce the problem):

1. Allow tracker to access photos, media, and files

2. Click on the add button on the lower right and enter plant details

3. Once the details are entered click on the checkmark onthe top left

4. Select the created profile.

5.Click on the three dots menu and select duplicate
