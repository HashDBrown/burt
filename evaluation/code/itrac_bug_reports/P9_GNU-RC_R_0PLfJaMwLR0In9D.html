Bug summary:

App crashes on making a new account

Observed Behavior (a description of what happened with the app):

When trying to create a new account named "Ben's" the app crashed unexpectedly.

Expected Behavior (a description of what you expected to happen):

I expected to be able to create a new account. The app shouldn't have crashed.

Steps to Reproduce (the steps to reproduce the problem):

1. Start the app

2. Tap the Create Accounts button (only Expenses and Income should be checked)

3. Tap the upper most plus symbol on the Accounts tool bar

4. Supply "Ben's" as the Account Name

5. Hit the save button

6. The app will crash

Additional information:

This bug was experienced from a cold start. Using the account name "Bens" works fine, so I suspect the apostrophe may be causing an issue.
