Bug summary:

The token does not appear when assigned a random seed

Observed Behavior (a description of what happened with the app):

When creating a random seed token, the token does not appear in the token list.

Expected Behavior (a description of what you expected to happen):

The name of the token will appear on the main menu page

Steps to Reproduce (the steps to reproduce the problem):

1. Open the application

2. Select the three dots in the upper-right corner

3. Select "Add Token"

4. Add a name to the token

5. Select "Next>"

6. Select the "Generate Random Seed" option

7. Select "Complete"
